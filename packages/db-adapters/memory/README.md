# @argusjs/db-memory

In-memory database adapter for testing and development.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/db-memory
```

## Usage

```typescript
import { MemoryDbAdapter } from '@argusjs/db-memory';

const db = new MemoryDbAdapter();
```
