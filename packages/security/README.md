# @argusjs/security-engine

Security engine providing anomaly detection, brute force protection, and device trust scoring.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/security-engine
```

## Usage

```typescript
import { DefaultSecurityEngine } from '@argusjs/security-engine';
```
