# @argusjs/dashboard

Next.js admin UI for managing users, sessions, and security settings.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/dashboard
```

## Usage

```typescript
// Start the dashboard dev server
pnpm --filter @argusjs/dashboard dev
```
