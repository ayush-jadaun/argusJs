# @argusjs/mfa-webauthn

WebAuthn/FIDO2 multi-factor authentication provider (passkeys).

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/mfa-webauthn
```

## Usage

```typescript
import { WebAuthnProvider } from '@argusjs/mfa-webauthn';
```
