# @argusjs/policy-hibp

Have I Been Pwned (HIBP) password breach check.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/policy-hibp
```

## Usage

```typescript
import { HIBPPolicy } from '@argusjs/policy-hibp';
```
