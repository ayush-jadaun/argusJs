# @argusjs/policy-zxcvbn

Password strength scoring using zxcvbn.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/policy-zxcvbn
```

## Usage

```typescript
import { ZxcvbnPolicy } from '@argusjs/policy-zxcvbn';
```
