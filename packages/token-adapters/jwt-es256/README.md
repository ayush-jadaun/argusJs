# @argusjs/token-jwt-es256

JWT ES256 (ECDSA) token provider.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/token-jwt-es256
```

## Usage

```typescript
import { ES256TokenProvider } from '@argusjs/token-jwt-es256';
```
