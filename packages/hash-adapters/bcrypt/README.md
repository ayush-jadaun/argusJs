# @argusjs/hash-bcrypt

bcrypt password hasher.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/hash-bcrypt
```

## Usage

```typescript
import { BcryptHasher } from '@argusjs/hash-bcrypt';

const hasher = new BcryptHasher({ rounds: 12 });
```
