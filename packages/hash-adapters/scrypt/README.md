# @argusjs/hash-scrypt

scrypt password hasher with no native dependencies.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/hash-scrypt
```

## Usage

```typescript
import { ScryptHasher } from '@argusjs/hash-scrypt';

const hasher = new ScryptHasher();
```
