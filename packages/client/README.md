# @argusjs/client

TypeScript SDK and React hooks for interacting with the ArgusJS server.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/client
```

## Usage

```typescript
import { ArgusClient } from '@argusjs/client';

const client = new ArgusClient({ baseUrl: 'http://localhost:3100' });
```
