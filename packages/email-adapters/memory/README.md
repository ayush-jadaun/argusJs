# @argusjs/email-memory

In-memory email provider for testing and development.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/email-memory
```

## Usage

```typescript
import { MemoryEmailProvider } from '@argusjs/email-memory';
```
