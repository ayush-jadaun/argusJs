# @argusjs/email-sendgrid

SendGrid email provider.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/email-sendgrid
```

## Usage

```typescript
import { SendGridEmailProvider } from '@argusjs/email-sendgrid';
```
