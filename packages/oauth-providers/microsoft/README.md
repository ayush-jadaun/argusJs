# @argusjs/oauth-microsoft

Microsoft OAuth provider.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/oauth-microsoft
```

## Usage

```typescript
import { MicrosoftOAuth } from '@argusjs/oauth-microsoft';

const microsoft = new MicrosoftOAuth({ clientId: '...', clientSecret: '...' });
```
