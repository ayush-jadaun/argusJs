# @argusjs/oauth-apple

Apple OAuth provider.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/oauth-apple
```

## Usage

```typescript
import { AppleOAuth } from '@argusjs/oauth-apple';

const apple = new AppleOAuth({ clientId: '...', clientSecret: '...' });
```
