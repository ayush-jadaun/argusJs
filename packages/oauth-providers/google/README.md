# @argusjs/oauth-google

Google OAuth provider.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/oauth-google
```

## Usage

```typescript
import { GoogleOAuth } from '@argusjs/oauth-google';

const google = new GoogleOAuth({ clientId: '...', clientSecret: '...' });
```
