# @argusjs/oauth-github

GitHub OAuth provider.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/oauth-github
```

## Usage

```typescript
import { GitHubOAuth } from '@argusjs/oauth-github';

const github = new GitHubOAuth({ clientId: '...', clientSecret: '...' });
```
