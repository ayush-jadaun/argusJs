# @argusjs/oauth-discord

Discord OAuth provider.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/oauth-discord
```

## Usage

```typescript
import { DiscordOAuth } from '@argusjs/oauth-discord';

const discord = new DiscordOAuth({ clientId: '...', clientSecret: '...' });
```
