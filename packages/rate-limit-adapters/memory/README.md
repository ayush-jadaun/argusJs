# @argusjs/ratelimit-memory

In-memory rate limiter for testing and development.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/ratelimit-memory
```

## Usage

```typescript
import { MemoryRateLimiter } from '@argusjs/ratelimit-memory';
```
