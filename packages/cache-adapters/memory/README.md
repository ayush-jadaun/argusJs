# @argusjs/cache-memory

In-memory cache adapter for testing and development.

Part of [ArgusJS](https://github.com/ayush-jadaun/argusJs) -- Enterprise-grade, fully pluggable authentication platform.

## Install

```bash
pnpm add @argusjs/cache-memory
```

## Usage

```typescript
import { MemoryCacheAdapter } from '@argusjs/cache-memory';
```
